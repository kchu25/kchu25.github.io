Proximal gradient descent

I often forget how to derive proximal gradient descent. So here we go.

Proximal gradient descent is for solving the problem of this form:

min⁡zf(z)+g(z)\min_{\bm z} f(\bm z)+g(\bm z)zmin​f(z)+g(z)

where fff is a differentiable but ggg is not. The idea of proximal operator is to do a quadratic approximation of fff at some point x\bm xx and solve that with ggg. We replace the hessian ∇2f\nabla^2 f∇2f by 1ηI\frac{1}{\eta}Iη1​I. We want to solve:

arg min⁡z f(x)+∇f(x)T(z−x)+12η∥z−x∥22+g(z)=arg min⁡z 12η∥z−(x−η∇f(x))∥22+g(z) \begin{align*} &\argmin_{\bm z}\, f(\bm x)+\nabla f(\bm x)^T(\bm z-\bm x)+\frac{1}{2\eta}\|\bm z-\bm x\|^2_2 + g(\bm z)\\ =&\argmin_{\bm z}\, \frac{1}{2\eta}\|\bm z-(\bm x-\eta\nabla f(\bm x))\|^2_2 + g(\bm z) \end{align*} =​zargmin​f(x)+∇f(x)T(z−x)+2η1​∥z−x∥22​+g(z)zargmin​2η1​∥z−(x−η∇f(x))∥22​+g(z)​

The proximal operator proxg,η(z)\text{prox}_{g,\eta}(\bm z)proxg,η​(z) is defined as the following:

proxg,η(x)=arg min⁡z12η∥z−x∥22+g(z) \text{prox}_{g,\eta}(\bm x) = \argmin_{\bm z} \frac{1}{2\eta}\|\bm z - \bm x\|^2_2 + g(\bm z) proxg,η​(x)=zargmin​2η1​∥z−x∥22​+g(z)

And proximal gradient descent is done by choosing an initial point x(0)\bm x^{(0)}x(0) and execute the following iterative procedure:

x(k)=proxg,ηk(x(k−1)−ηk∇f(x(k−1))),k=1,2,3... \bm x^{(k)} = \text{prox}_{g,\eta^k}(\bm x^{(k-1)}-\eta^k\nabla f(\bm x^{(k-1)})),\quad k=1,2,3... x(k)=proxg,ηk​(x(k−1)−ηk∇f(x(k−1))),k=1,2,3...

There are many theoretical properties to show that this basically behaves like gradient descent. I'd skip that for now.
