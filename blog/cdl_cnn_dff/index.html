Understand convolutional dictionary learning vs convolutional neural network in one picture

In the talk (at 21:57) given by Brendt Wholberg, the following slide illustrate the difference between a convolutional dictionary learning model (CDL) and a convolutional neural network (CNN).

Slide from Brendt Wholberg's talk on Convolutional Sparse Representations for Imaging Inverse Problems at NC state ECE 2018.

Of course, the symbol ∗*∗ is the convolution operator. Here, the deconvolutional network is equivalent to the tranlational invariant (sparse) representation. In another word, the deconvolutional network is a CDL.

In CNN, the filters are convolved with the input signal (image) to generate a feature representation.

In CDL, the filters are convolved with the code vectors to represent the input signal.

References

  • Matthew D and Krishnan Zeiler, Deconvolutional networks, 2010.
  • Cristina and Wohlberg Garcia-Cardona, Convolutional dictionary learning: A comparative review and new algorithms, 2018.
